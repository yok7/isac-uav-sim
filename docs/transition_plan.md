# 准研究生过渡期执行路线图（4-8月）

## 目标对齐

- 跑通通感一体化仿真闭环：波形 -> 信道 -> 阵列接收 -> DOA估计 -> 评估
- 完成经典DOA（Beamforming/MUSIC/ESPRIT）与1-2种前沿DOA算法对比
- 给出通信-感知权衡曲线（Pareto前沿）
- 形成可投稿雏形的技术报告

## 周粒度计划

### 第1-2周（环境和基础）

- 完成 Python 环境和代码仓库结构
- 跑通 Sionna 官方样例（至少1个）
- 完成 8阵元 ULA 方向图绘制
- 输出：阅读笔记 v1 + 方向图 + 环境自检截图

### 第3-4周（经典DOA）

- 实现并验证 Beamforming / MUSIC / ESPRIT
- 在统一仿真数据上做初步精度和耗时对比
- 输出：经典算法对比图（角谱图 + RMSE曲线）

### 第5-8周（前沿DOA与资源权衡）

- 选 1-2 个前沿算法进行复现
- 建立通信资源与感知资源可调参数实验
- 绘制 BER(or Rate) vs RMSE 的权衡曲线
- 输出：前沿算法复现记录 + 权衡分析图

### 第9-12周（低空场景与整合）

- Sionna RT 构建 LoS 与 NLoS 两类场景
- 整合完整链路并做端到端评估
- 输出：场景对比结果 + 完整实验配置

### 第13-16周（报告与打磨）

- 固化实验配置与复现实验脚本
- 完成技术报告初稿并循环修改
- 输出：技术报告初稿 + 可复现代码索引

## 每周固定交付模板

- 一个公式：本周核心数学表达
- 一张图：本周关键实验结果
- 一句话结论：本周最重要发现
